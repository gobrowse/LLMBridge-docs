# Supported Providers

The current prototype contains built-in metadata for these providers:

- `openai`
- `anthropic`
- `groq`
- `openrouter`

## Important note

Built-in support here means the code knows the default endpoint shape and auth-header style for those providers. It does not mean every model or every edge case has been validated in these docs.

## Environment variable examples

- `OPENAI_API_KEY`
- `ANTHROPIC_API_KEY`
- `GROQ_API_KEY`
- `OPENROUTER_API_KEY`
