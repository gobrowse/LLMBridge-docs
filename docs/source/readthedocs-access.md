# Read the Docs Access and Authentication

## Short answer

Yes, Read the Docs offers authentication and access controls, but the exact level depends on the platform plan and the privacy settings you use.

## What matters most

- access control is handled at the project and version level
- it is not designed as per-page or per-path authentication
- private documentation and advanced sharing controls are associated with Read the Docs for Business

## Practical interpretation for this project

If this documentation stays fully public, the community Read the Docs flow is enough.

If you later want restricted docs for only some users, you should plan around:

- private or public project/version privacy settings
- Business-plan sharing features
- organization/team access if the docs become non-public
