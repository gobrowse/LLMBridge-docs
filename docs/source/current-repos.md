# Current Repository Layout

## Prototype repo

The prototype repo is the active working repository. It contains the current source-first implementation.

## Pip-oriented repo

A separate sibling repository was created to prepare a cleaner package layout under `src/aimgmt/`.

## Documentation repo

This repository exists only for documentation and Read the Docs hosting.

## Why the split exists

The split allows packaging and documentation work to move forward without forcing the prototype repo to pretend it is already a finished library.
